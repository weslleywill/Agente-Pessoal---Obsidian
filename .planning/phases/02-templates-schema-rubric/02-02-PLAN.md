<interfaces>
Templater syntax (v2.19.3):
- `<% tp.date.now("YYYY-MM-DD") %>` — data atual formatada
- `<% tp.file.title %>` — nome do arquivo sem extensão
- `<%* ... %>` — bloco executável (sem output)
- `<% tp.file.include("[[frag-nome]]") %>` — injeta fragment inline
- `<% tp.date.now("dddd", 0, tp.file.title, "YYYY-MM-DD") %>` — dia da semana a partir do nome do arquivo (parse retroativo)

- Daily: `01 - Diário/YYYY/MM-MMMM/YYYY-MM-DD` → TPL-Diario.md
